# BARMM EBDM Consolidated Dashboard

Static dashboard snapshot generated 2026-07-17.

## Files

- `index.html` - shareable static dashboard
- `BARMM_EBDM_Consolidated_Dashboard_20260717.xlsx` - Excel dashboard workbook
- `data/` - sanitized summary CSV and JSON tables used by the dashboard

## Source Scope

This package is built from the four provincial dashboard workbooks and the three analysis overview workbooks supplied in `output/chatgpt_project_setup_20260708/04_province_workbooks`.

Maguindanao del Sur has no matching analysis workbook in the supplied file set, so its coverage comes from `maguindanaodelsur_provincial_dashboard.xlsx` only.

The source coverage check is visible in the dashboard because the analysis overview files and provincial dashboard scorecards report different facility counts for several provinces.
